# Vante Bots

Merhaba,

Eğer ilgileniyorsanız, detayları konuşmak için Discord üzerinden benimle iletişime geçebilirsiniz. Discord kullanıcı adım: [@q7x](https://vante.dev/discord)
İlginiz için teşekkür ederim!
